A mobile app for the vulnerable during a crisis to pair up with volunteers to get help with their shopping. GraphQL MongoDB Dart Flutter

React front-end for news aggregation web app, utilising data from my News from the Northcoders REST API. React API CSS JSX

React web application news site with a simulated login system. When 'logged in', users can post an article or a comment, and delete anything they have posted. Users can sort articles, up and down vote articles and comments. App also has pagination to click through pages of articles or comments. Future features would include: Ability to register a new user and login with that user's details. Authentication using JWT on the back-end.

News from the Northcoders - RESTful API serving up a range of endpoints (MVC architecture). Node.js Express Knex PostgreSQL

A RESTful API, created using MVC architecture (model-view-controller), serving up a number of endpoints for GET, POST, PATCH, and DELETE requests for news/articles/users information. The API also has pagination, limiting the number of article or comment requests to 10. The API was created using Node.js, Express JS, paired with Knex.js - an SQL query builder for Postgres, which was also used. Future improvements could include authentication using JWT to allow login.

React web application, displaying earthquake data. React API CSS JSX

A single page application using real-time web API calls to bring in current data on earthquakes. Quakr has a responsive UI and a drop-down filters menu. Each pointer on the map has location, time and a link. The map visualiation was created using React-Leaflet - a third party implementation of Leaflet JS. Using this package brought some challenges, as the React docs were not as comprehensive, so I had to search for additional resources.

Katie's Bookshelf - React drag 'n drop books project. React API CSS JSX

A bookshelf web app created using React and data from the Google Books API. Books can be moved from the "want to read" to the "finished" bookcase using drag and drop functionality, created with Atlassian's Beautiful Drag N Drop package. As a relatively new user to React when creating this project, I set myself the challenge of using a new third-party package to accomplish something different. I love the drag 'n' drop effect! To improve this project, I would work on the UI design, and allow users to add multiple bookcases - but for now this is a personal project.
